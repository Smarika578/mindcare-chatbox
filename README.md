# mindcare-chatbox
mindcare-chatbox🧠💙

Overview

MindCare Chatbot is a voice-enabled AI mental health assistant built using React.js. It allows users to communicate through text or voice and receive AI-generated responses. The application also supports text-to-speech functionality for an interactive user experience.

Features

- 🎤 Speech-to-Text (Voice Input)
- 🔊 Text-to-Speech (Voice Output)
- 💬 AI-Powered Chat Responses
- ⚡ Quick Suggestion Buttons
- 📱 Responsive User Interface
- 🧠 Mental Wellness Support Conversations

Technologies Used

- React.js
- JavaScript (ES6)
- OpenAI API
- Web Speech API
- HTML5
- CSS3

Project Workflow

1. User enters a message through text or voice.
2. The message is sent to the AI model.
3. AI generates a response.
4. The response is displayed in the chat window.
5. Text-to-Speech converts the response into audio.
